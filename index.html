Mes compétences

Front-end

HTML

Langage standard utilisé pour créer et structurer les pages web. Il sert de base pour organiser le contenu d'un site web en définissant des éléments comme les titres, paragraphes, images, liens, boutons, formulaires, etc.

CSS

Utilisé pour décrire la présentation et le design des pages web construites avec HTML. Il contrôle l'apparence visuelle des éléments pour offrir une expérience utilisateur attrayante et cohérente.

JAVASCRIPT

Utilisé principalement pour ajouter de l'interactivité et du dynamisme aux pages web.

BOOTSTRAP

Framework front-end open-source qui facilite la création de sites web, accélère et simplifie le développement en fournissant des composants prédéfinis, des grilles flexibles et des styles CSS, ainsi que des plugins JavaScript.

Back-end

JAVA

Langage robuste, sécurisé et polyvalent, utilisé dans un large éventail de secteurs pour créer des applications fiables et performantes.

Base de donnée

SQL ou NoSQL, les bases de données sont des systèmes conçus pour stocker, organiser, gérer, et récupérer de grandes quantités d'informations de manière efficace.

SPRING BOOT

Facilite le développement d'applications Java, notamment les applications web. Il propose une approche simplifiée et rapide pour configurer et déployer des applications.

POSTMAN

Il fournit une interface conviviale permettant aux développeurs d'interagir avec des APIs de manière simple et efficace, sans avoir à écrire du code complexe pour tester les requêtes.

Environnement et outils

VSCODE

Éditeur de code source développé par Microsoft, populaire pour son interface légère et sa puissance. C'est un outil de développement multi-plateforme qui fonctionne sous Windows, macOS et Linux.

FIGMA

Figma est un outil de conception graphique et de prototypage basé sur le web, utilisé principalement pour le design d'interface utilisateur (UI) et le design d'expérience utilisateur (UX).

METHODE AGILE

Approche de gestion de projet, principalement utilisée dans le développement de logiciels, qui privilégie la flexibilité, la collaboration et la livraison rapide.

DEVOPS

Approche de gestion des processus de développement et des opérations informatiques qui vise à améliorer la collaboration et l'efficacité entre les équipes de développement (Dev) et les équipes des opérations (Ops).
